这个系列的第二篇从上学期鸽到现在。。。

线性代数最基本的问题就是解线性系统了，但是这也是个很不trivial\text{trivial}trivial的问题。

求解线性系统，主要有两类方法，直接求解（分解）或者迭代方法。对于大型稀疏矩阵而言，两种方法各有优势，都有不少深入的文献和代码可以学习。这次简要介绍求解线性系统的迭代方法，直接方法留到这个系列的第三篇（又不知道会鸽到什么时候了，大悲）。

迭代法是一个很大的话题，也有一些专著，比如Yousef Saad\text{Yousef Saad}Yousef Saad(GMRES\text{GMRES}GMRES方法的发明者)的书Iterative Methods for Sparse Linear Systems\text{Iterative Methods for Sparse Linear Systems}Iterative Methods for Sparse Linear Systems。这次不仅参考了这本书，同时也参考了徐树方的数值线性代数。
